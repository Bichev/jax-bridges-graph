## ğŸ”§ Common Issues & Solutions

### Issue: "OPENAI_API_KEY not found"

**Solution:** Make sure you created `.env` file with your API key:
```bash
OPENAI_API_KEY=sk-proj-xxxxx
```

### Issue: "Failed to load data"

**Solution:** Run the analysis first:
```bash
npm run analyze
```

### Issue: "Module not found" errors

**Solution:** Reinstall dependencies:
```bash
rm -rf node_modules package-lock.json
npm install
```

### Issue: Analysis taking too long

**Solution:** Use sample mode to test with fewer businesses:
```bash
npm run analyze:sample
```

### Issue: Rate limit errors from OpenAI

**Solution:** The script has built-in retry logic. If you hit limits:
1. Wait a few minutes
2. The script saves progress, so you can resume
3. Consider upgrading your OpenAI plan
